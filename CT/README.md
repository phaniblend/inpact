## Question Structure

Each question includes:
- **Title** - Descriptive name
- **Time** - Expected completion time
- **Tests** - What concepts this tests
- **Challenge** - Specific, real-world problem (NOT generic placeholders)
- **What interviewers look for** - Complete, working code solution

## Difficulty Levels

- **Junior (25 questions)**: 0-2 years experience, 5-15 min each
- **Mid (35 questions)**: 2-5 years experience, 15-30 min each  
- **Senior (25 questions)**: 5+ years experience, 30-60 min each
- **Lead (15 questions)**: 7+ years experience, 60-120 min each

## Key Principles

✅ **GOOD**: "Create a traffic light component with three circles (red, yellow, green). When you click a button, cycle through the lights..."

❌ **BAD**: "// Implement conditional rendering"

Every question is a REAL, SPECIFIC problem with complete working solutions.
