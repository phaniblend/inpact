## What is INPACT?

INPACT is a problem-based React learning engine. Instead of watching videos or reading docs, you build real components ‚Äî guided by Socratic questioning, instant feedback, and analogous examples.

Every problem is one concept. One concept at a time. No hand-holding, no spoon-feeding ‚Äî just you, the editor, and the right question at the right moment.

## Stack

- React 18 + Vite
- Zero runtime AI calls ‚Äî problems are authored once, served free forever
- Custom syntax-highlighted code editor (no Monaco, no npm bloat)

## Run locally

```bash
git clone https://github.com/YOUR_USERNAME/inpact-learn
cd inpact-learn
npm install
npm run dev
```
